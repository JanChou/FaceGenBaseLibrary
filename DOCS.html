FaceGen Base Library Documentation

Requirements

  • Windows:
    • Visual Studio {2008,2010,2012,2013} Professional or higher.
  • OS/X:
    • Xcode {4,5}
      • Menu: Xcode -> Preferences -> Downloads -> Command Line Tools
      • Add to path: $ PATH=$PATH:/Applications/Xcode.app/Contents/Developer/usr/bin/
  • Ubuntu:
    $ sudo apt-get update
    $ sudo apt-get install build-essential libc6-dev-i386 g++-multilib make

Compile

Path in which you have placed this library:

~fgbl

Solution files for Visual Studio:

~fgbl/source/VisualStudio2008.sln
~fgbl/source/VisualStudio2010.sln
~fgbl/source/VisualStudio2012.sln
~fgbl/source/VisualStudio2013.sln

Makefiles for OS X / Ubuntu (use 'make' with '-f' flag to select makefile):

~fgbl/source/Makefile.osx.gcc.64.debug
~fgbl/source/Makefile.osx.gcc.64.release
~fgbl/source/Makefile.ubuntu.gcc.64.debug
~fgbl/source/Makefile.ubuntu.gcc.64.release

Use

Set up the path ('>' windows. '$' osx,ubuntu) for 64bit release version (VS2012 for Windows):

> PATH=%PATH%;~fgbl\bin\win\vs12\64\release
$ PATH=$PATH:~fgbl/bin/osx/gcc/64/release
$ PATH=$PATH:~fgbl/bin/ubuntu/gcc/64/release

View all available commands:

$> fgbl

Run all automated tests:

$> fgbl test all

View all manual (interactive) test categories (GUI currently only supported on Windows):

$> fgbl testm

Rebuild makefiles, solution files and project files by scanning existing source code files:

~fgbl$> fgbl cons
